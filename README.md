## Password Generator by Desaign Studio

A password generator is software program that takes input from a random number generator and automatically generates a password.

<img src="./SS1.png" alt="Screen Shot" width="500">


### Features

- Once password generates, you can click to copy the password.
- Exclude similar characters ( e.g. i, l, 1, L, o, 0, O )
- Exclude ambiguous characters ( e.g. { } [ ] ( ) / \ ' " ` ~ , ; : . < > )
